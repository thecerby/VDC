VDC : Tenant dashboard

Created by Stephan Oomen, last modified on 20 jan., 2022

The tenant dashboard is available when you you enable the permission Dashboard.

Within the dashboard we have a list of widgets that can be added, by enabling edit mode, you can add widgets, pages and fallback to default.

The default dashboard contains the following widgets:

  • Vehicle general statistics

  • Vehicle status

  • Vehicle location statistics

Vehicle general statistics

Requires vehicles permission

This shows a number of vehicles processed within the specified date range.
Created => Number of vehicles created in the date range
Company Stock => Number of vehicles put on company stock in the date range
Sold inland => Number of vehicles sold in the date range
Sold foreign country => Number of vehicles exported in the date range

Vehicle status

Requires vehicles permission

This shows a list of statuses and the number of vehicles in that status.

Vehicle location statistics

Requires vehicles permission

This shows a graph how many vehicles are on the tenants own location or (planned) at AFC that are not yet picked up.
