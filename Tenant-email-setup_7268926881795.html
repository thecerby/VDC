VDC : Tenant email setup

Created by Stephan Oomen, last modified on 22 feb., 2022

In order to enable mail support for a tenant, the following things needs to be taken care of:

  1. Configure settings for the tenant

  2. Configure mail portal in Portal Server

  3. Enable feature/permissions in the backoffice.

Add settings

Add the settings into the tblMailSettings table for each mail setting.

Configure mail portal in Portal Server

Now you need to make a shortcut for the EmailPortal in the Portal Server. Make sure to start it.

Enable feature/permissions in the backoffice.

Make sure the feature and settings are enabled for the tenant so it will be used.
Double check the Settings in the details of the tenant. There should be no ‘red’ blocks.
