VDC : Request For Sell

Created by Kavitha Thangappan, last modified on 10 nov., 2021

Customer who wants to sell the vehicle submits request to this endpoint.

System will update SoldOn date by today SoldBy with current username. It also checks whether customer type is private or business. If the customer type is business, system updates vehicle Vatcheck information from customer vat details like latestVatcheck, IsVatcheckNumberCorrect and vatId.

System updates CustomerId of vehicle with the customerId provided as input parameter and the authorizationId as well.

Then system will freeze the vehicle.

For using this api, you need to be authorized.
