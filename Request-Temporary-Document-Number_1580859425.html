VDC : Request Temporary Document Number

Created by Kavitha Thangappan, last modified on 10 nov., 2021

Customer who wants temporary document number for the specific vehicle sends request to this endpoint.

Here system will check vehicle on company stock. If not system throws VehicleNotOnStock exception.

This api method will call Rdwservice - RequestforTdn method passing vehicleId, userId as input parameters. If the request is succeeded , it will return DocumentNumber, AttendenceCode1, AttendenceCode2 as response.

For using this api, you need to be authorized.

Authentication

Please refer to this page. Authentication APIs - VDC - Confluence (atlassian.net)

Authorization

This requires api permission Pages_Vehicles and feature permission RdwRequestTdnFeature
