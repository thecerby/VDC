VDC : ATD / TDN

Created by Stephan Oomen on 27 dec., 2021

URL:       /VWE/wsATD.asmx

 

Workflow

ATD has a series of workflow to complete the request at RDW. The reason for this workflow is to ensure we return the temporary codes from RDW that are retrieved via an RDW web portal which is not 100% certain to deliver the data within a standard HTTP request.

Send ATD request

Soap Message

versturen

Help URL

/VWE/wsATD.asmx?op=versturen

Method (Verb)

POST

Headers

SOAPAction: "http://www.xmlMode.net/atd/versturen"
Content-Type: text/xml; charset=utf-8

Parameters

  • kenteken (required)

  • meldcode (required)

Request body

XML format

<soapenv:Envelope xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/" xmlns:atd="http://www.xmlMode.net/atd/">
   <soapenv:Header/>
   <soapenv:Body>
      <atd:versturen>
         <atd:xml_in_versturen>
            <atd:client>
               <atd:authenticatie>
                  <atd:reseller>
                     <atd:klantnummer></atd:klantnummer>
                     <atd:inlognaam></atd:inlognaam>
                     <atd:wachtwoord></atd:wachtwoord>
                  </atd:reseller>
                  <atd:eindgebruiker>
                     <atd:klantnummer></atd:klantnummer>
                     <atd:inlognaam></atd:inlognaam>
                     <atd:wachtwoord></atd:wachtwoord>
                  </atd:eindgebruiker>
               </atd:authenticatie>
            </atd:client>
            <atd:voertuig>
               <atd:kenteken></atd:kenteken>
               <atd:meldcode></atd:meldcode>
            </atd:voertuig>
         </atd:xml_in_versturen>
      </atd:versturen>
   </soapenv:Body>
</soapenv:Envelope>

Responses

200: XML format

<soap:Envelope xmlns:soap="http://schemas.xmlsoap.org/soap/envelope/" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xmlns:xsd="http://www.w3.org/2001/XMLSchema">
   <soap:Body>
      <versturenResponse xmlns="http://www.xmlMode.net/atd/">
         <xml_out_versturen>
            <loginresult>
               <code></code>
               <melding> </melding>
            </loginresult>
            <voertuig>
               <transactieid></transactieid>
               <documentnummer></documentnummer>
               <tenaamstellingscode1></tenaamstellingscode1>
               <tenaamstellingscode2></tenaamstellingscode2>
            </voertuig>
            <transactiemeldingen>
               <statuscode></statuscode>
               <foutmeldingen></foutmeldingen>
           </transactiemeldingen>
         </xml_out_versturen>
      </versturenResponse>
   </soap:Body>
</soap:Envelope>
