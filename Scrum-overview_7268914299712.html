Definition Of Done

We agreed on 20 dec. 2021 to accept the following criteria:

  • Acceptance criteria within a story are functional ‘tests’ to accept the story

  • Breakdown is provided by a developer to understand what needs to be done

  • Code changes are reviewed by a Pull Request

  • Code changes are tested by a peer developer

  • Code is passing the build process

  • User description is provided how to install or use the new story.

    • For example 'kenmerken/features' addition needs to be explained how this works.

  • The Product Owner accepts the story

Ceremonies

The following ceremonies are taken place:

  • Sprint of 2 weeks

  • Standups: 6 times on Tuesday, Wednesday and Thursday

  • Sprint review, at Friday 10:00 or 13:00

  • Sprint refinement & planning, at Monday 10:00 or 13:00

User Stories

  • Description of context (background information) why this is needed

  • Desired solution for resolving the needs

  • Acceptance criteria to accept the user story

  • Refine each user story

  • Definition is written in confluence

  • Estimates are done in ‘estimated hours’

  • Comply to priority order, then to order of the backlog

Example description of User Story:

As a user I want to have additional document lines to upload the following documents:

  • Document 1

  • Document 2

  • Document 3…

And when I have scanned the document it automatically uploads it to the Electronice Vehicle File via e.g. FTP
Context
Op dit moment worden van alle auto's dossier gebundeld, ingescand en vervolgens handmatig geupload via een sharepoint per gebruiker. Dit setje documenten (EVD) wordt altijd geupload onder de kop vervoersverklaring, nadat hert voertuig is uitgegeven. In de nabije toekomst moeten we geautomatiseer het setje documenten weer uit splitsen in losse documenten zoals oa legitimatie, cmr etc. Voor nu zou het veel tijdwinst opleveren als we al deze ingescande dossier automatisch geupload kunnen krijgen onder de kop vervoersverklaring. Idealiter maken we een nieuwe documentregel aan met de naam EVD

Acceptatie Criteria

  • op een loactie geplaatste documenten worden automatisch geupload

  • onder de kop EVD wordt t dossier geupload.
