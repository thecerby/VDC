VDC : Bulk processing

Created by Stephan Oomen on 27 dec., 2021

This process requires the permission Pages.Vehicles.BulkProcessVehicles

Within the RegistratieXpert it is possible to bulk process existing vehicles. With using an template, values can be updated and processed through RegistratieXpert.

Currently the bulk processing supports the following features:

  • Create customer if not already exists

  • Assigning customer to vehicle

  • Validate VAT number if supplied

Validating VAT number requires the feature App.VatCheckFeature

  • Mark vehicle as sold when it is a domestic sale

  • Assign lamination codes to vehicle.
