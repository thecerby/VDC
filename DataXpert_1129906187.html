VDC : DataXpert

Created by Stephan Oomen, last modified on 27 dec., 2021

The DataXpert, also known as VDC.Services in the repository requests information of an vehicle configured per account. There is an Swagger Documentation available for registered accounts.

This project is found in VDCServices repository.

Vehicles

Information

Vehicle information can be requested through the API, based on the acount it returns the vehicle information. This information set is given by Cartalk. VDC and Cartalk has set up the following service:

  • lp

  • jet

  • foto

Each of these sets contains information of a vehicle. An account for the service can constist on 1 or more of these services.

Registration

Vehicle registrations basically does the same thing as RX does, but using an API instead. It requires the account to be coupled to the SubscriberId/TenantId. Also, it includes the "information" part if the services are configured.

Insurance Check (ASR Detect)

When requesting an green card, we need to check if the person that will drive the car is not blacklisted by the insurance company. It will call the Friss Service which is IP based protected if the person is eligle to get the green card. If not, it will send an email to ASR.
