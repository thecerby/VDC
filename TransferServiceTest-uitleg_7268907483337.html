In this article i will explaine to you how to use TransferServiceTest.

OrbServiceTest needs to be runned before you runned this test every time you want to run this test.

 Instructions

In this project we are gonna test TransferServiceTest and see if the program works te way we want it to.

You will have to do OrbServiceTest first before you can run this test otherwise it wont work.

there is a link to OrbServiceTest ad the bottom of this document, there is explained what we need to do for OrbServiceTest to work with TransferServiceTest.

First we are gonna look ad a few lines of code from TransferServiceTest, the lines of code we are gonna change are mention and show down below.

        {
            SupplyingRdwCompany = 1912,
            ReceivingRdwCompany = 1913,
            Reference = string.Format("TRANSFER:RX:IntegrationTest_{0}", Guid.NewGuid()),
            LicensePlate = "GTVD48",
            ReportCode = "4548",
            WaitForApprovalAcceptance = true,
            Counter = new VehicleCounter
            {
                HasCounter = true,
                Number = 1001,
                Unit = RdwVehicleCounterUnit.Kilometers
            }

The 2 lines underneath here you will need to change.

            LicensePlate = "GTVD48",
            ReportCode = "4548",

The information you will need to give to these 2 lines are the same information you gave in OrbServiceTest.

how to make the program work

  1. If you cleared and filled in those two lines of code i mentioned, you can go ahead and run the test.

2. If the test is runned and there is a green ball the test works and you have completed the test, if there is a f f red ball the test failed.

possible things that could have gone wrong if youre test failed

If you had a red ball you can read this for an option to make it work.

Im gonna give a list to check if you did these things.

1. Did you give the exacte same information to TransferServiceTest as you gave to OrbServiceTest.

2. Did you run the OrbServiceTest before you ran this test.

3. Did you check if there are some missing parts in the code where something should stand.

4. Did check a other licence plate if that worked.

If you tested youre TransferServiceTest you will need to do another OrbServiceTest run because you can only do it ones, afther that you will need to send another another one.
