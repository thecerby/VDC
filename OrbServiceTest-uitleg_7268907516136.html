 Instructions

In this project we are gonna test OrbServiceTest as prepration for TransferServiceTest.

First we are gonna look ad a few lines of code from OrbServiceTest, the lines of code we are gonna change are mention and show down below.

you can only use one license plate 1 time a day so afther that you will need to change the information you give to the system. But if you have already safed the license plate into the server you can use it more then ones.

how to make the program work

1. You can go ahead and cleared the 5 lines of code, afther that you can go ahead and have a look ad the f f excel sheet, as i mentioned earlier you will have to choose a row from the horizontal side and aline the f f f information that is given to you in the excel sheet and the lines of code that you have seen in this f f f f f f f document.

2. If you filled in those 5 lines with the right information you can go and run the test.

4. If the test is runned and there is a green ball the test works and you can go ahead and go to f f f f f f f TransferServiceTest. If there is a red ball the test failed.

possible things that could have gone wrong if youre test failed.

If you hafe a green ball you can skip this part and go on to TransferServiceTest.

If you hafe a red ball you can read this for an option to make it work.

Im gonna give a few options to check if you did something wrong.

1. Did you give the exacte same information in OrbServiceTest as there in the excel sheet.

2. Did you check if there are some missing lines or code or somewhere else or if something else changed.

REMEMBER. You can only use a licensplate max 1 time a day, every night it will reset, but if the licence plate is already saved in the server then you can use it more often.
