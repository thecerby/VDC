VDC : Authentication APIs

Created by Kavitha Thangappan on 27 okt., 2021

Customer who wants access to access application endpoints needs to be authenticated first.

Property

Value

Endpoint

​/api​/TokenAuth/Authenticate

Method

POST

Headers

Accept: application/json 

Content-Type: application/json 

Field Parameters

  • userNameOrEmailAddress- string- (required) 

  • password- string- (required) 

  • twoFactorVerificationCode- string- (required) 

  • rememberClient- boolean- (required) 

  • twoFactorRememberClientToken- string- (required) 

  • singleSignIn- boolean- (required) 

  • returnUrl- string- (required) 

  • captchaResponse- string- (required) 

Input

{
  "userNameOrEmailAddress": "string",
  "password": "string",
  "twoFactorVerificationCode": "string",
  "rememberClient": true,
  "twoFactorRememberClientToken": "string",
  "singleSignIn": true,
  "returnUrl": "string",
  "captchaResponse": "string"
}

Output

if request is successful , api returns below response. Response status 200

  "userId": 0,

    "string"
  ],
