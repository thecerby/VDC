VDC : General overview

Created by Stephan Oomen, last modified on 28 mei, 2024

Below you’ll find general information about the RegistratieXpert in the current situation.
A representation of the services you will find below

VDC UI (2.0)

The UI interface is built in Angular 11+ and is setup using Aspnetzero. Each main feature is set up into modules so the starting bundle stays small. From the UI you can log in as host or tenant. Each tenant has it’s own name and can be set up separately.

The UI interface is hosted on a Azure App Service named “regxpert-ui”

Consumers:

  • All tenants

  • Host users

VDC API (2.0)

The API interface is built with the UI in mind, all services are REST based and boilerplated from Aspnetzero. Each main feature is set up into separate service. Here the UI connects to the API but it is also possible to connect a third-party software to the API.

The API is hosted on a Azure App Service named “regxpert-api”

Consumers:

  • VDC UI

VDC Services

The services interface is used to public use before 2.0 was built. Here this contains the ability to retrieve vehicle information or register a new vehicle. The service has a separate login (basic) to authenticate. This user is then connected to a tenant (if registration is required).

Also the service is whitelisted to do ASR check in behalf of Silverlight or the 2.0 version.

The API is hosted on a Azure App Service named “vdc-services”

Consumers:

  • VDC API

  • RE Services

  • BCA

Silverlight

This (old) version of the RE has the same abilities as the VDC UI but is deprecated due to the lifetime of Silverlight.

Consumers:

  • All tenants (if not using VDC UI)

Kentekencardreader

This application allows you to register vehicles through an application that reads a licenseplate card.

This is connection to the old authentication mechanism.

Consumers:

  • All tenants if using this software

BCA Service

This API is based on the VWE services, this was created due to the migration from VWE to VDC. This is initially explicitly built for BCA. There is an outstanding issue to let BCA use the VDC Services.

Consumers:

  • BCA
