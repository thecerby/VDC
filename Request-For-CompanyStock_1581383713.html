VDC : Request For CompanyStock

Created by Kavitha Thangappan, last modified on 11 nov., 2021

Customer who wants to put the vehicle on stock submits request to this endpoint.

System checks whether vehicle is on company stock, if not it throws user friendly exception VehicleNotOnStock

System calls RdwService - RequestforOrb method by passing VehicleId, RequestNewLicensePlateCard,RequestCounterReport,KeepRegistrationCertificate and additionalMailaddresses as input parameters. If result is not succeeded system throws error message.

If the result is successful system updates vehicle oncompany stock to true , date by currentdate and time and the supplierId.

For using this api, you need to be authorized.
