VDC : Vehicles

Created by Stephan Oomen on 01 dec., 2021

Within the RegistratieXpert it is possible to create vehicle records. The following information can be provided upon creation of the file.

  • License Plate

  • VIN number

  • Report Code

  • Counter Position

  • License Plate Document Number

  • Attendance Codes

Available workflow actions provided as features:

  • RDW ORB - Company Stock Registration

  • RDW TDN - Request Temporary Document

  • RDW OREH - Export to foreign country

  • RDW Transfer - Transfer to other company

  • Sell to customer

  • Insurance - Request insurance document for 14 days
