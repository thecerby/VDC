VDC : General information

Created by Stephan Oomen on 27 dec., 2021

Introduction

This document describes technical information for the VDC VWE Services It is meant to be read by developers that are implementing these services. The services are a look-a-like on the VWE services so parties from VWE can easily connect to VDC.

The API is accessible through SOAP protocol. Everything is handled through the SOAP envelope body.

Authentication

In order to access the API you need to be authorized using credentials. These credentials will be provided by VDC or supplier.

 

The authorization can be given in the body on the <client> node.

<client>
    <authenticatie>
        <eindgebruiker>
            <klantnummer>?</klantnummer>
            <inlognaam>?</inlognaam>
            <wachtwoord>?</wachtwoord>
        </eindgebruiker>
    </authenticatie>
    <referentie>
        <info></info>
    </referentie>
</client>

When the credentials are not given or invalid, HTTP Status 401 will be returned.
