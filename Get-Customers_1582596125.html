VDC : Get Customers

Created by Kavitha Thangappan, last modified on 11 nov., 2021

Customer who wants to fetch list of customer details filtered based on filter condition and sort by field sends request to this endpoint.

System fetches all customer details through customer repository based on the filter conditions provided SearchName, CustomerNmber, Name , Customer type - Private or Business and Customer category Buyer or Seller.

System returns the result as paged result.

For using this api, you need to be authorized.

Authentication

Please refer to this page. Authentication APIs - VDC - Confluence (atlassian.net)

Authorization

This requires api permission Pages_Vehicles
