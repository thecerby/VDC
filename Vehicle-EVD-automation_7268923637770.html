VDC : Vehicle EVD automation

Created by Stephan Oomen, last modified on 04 feb., 2022

Every friday e.o.d. a job runs for processing documents found it the “Scanwerk” folder of the VDC Sharepoint environment. It will retrieve the file and add to a EVD document on the vehicle. After uploading, the document is then moved to a central folder named “Processed”.
