VDC : Get Customer Documents

Created by Kavitha Thangappan, last modified on 11 nov., 2021

Customer who wants to fetch customer documents sends request to this endpoint.

System fetches all Customer documents associated with the customer id from database through customer document repository.

For using this api, you need to be authorized.

Authentication

Please refer to this page. Authentication APIs - VDC - Confluence (atlassian.net)

Authorization

This requires api permission Pages_Vehicles

Value

Endpoint

​/api​/services​/app​/Customer/GetCustomerDocuments

Authorization
