VDC : Update Customer

Created by Kavitha Thangappan, last modified on 11 nov., 2021

Customer who wants to update existing customer sends request to this endpoint.

System updates customer detail into database with the values provided in the input parameter.

For using this api, you need to be authorized.

Authentication

Please refer to this page. Authentication APIs - VDC - Confluence (atlassian.net)

Authorization

This requires api permission Pages_Customers and feature permission Pages_Customers_Edit

 

Property

Value

Endpoint

​/api​/services​/app​/Customer/UpdateCustomer

Method

POST

Authorization

Token {token from Authorized User}

Headers

Accept: application/json 

Content-Type: application/json 

Field Parameters

  • CustomerNumber- string- (required) 

  • Name- string- (required) 

  • SearchName- string- (required) 

  • VatId- string- (required) 

  • Address - Street - string- (required) 

  • Address - HouseNumber- string- (required) 

  • Address - HouseNumberAddition- string- (optional) 

  • Address - PostalCode- string- (required) 

  • Address - City- string- (required) 

  • Address - CountryCode- string- (required) 

  • ContactAddress - Street - string- (optional) 

  • ContactAddress - HouseNumber- string- (optional) 

  • ContactAddress - HouseNumberAddition- string- (optional) 

  • ContactAddress - PostalCode- string- (optional) 

  • ContactAddress - City- string- (optional) 

  • ContactAddress - CountryCode- string- (optional) 

  • Contact - Email- string- (optional) 

  • Contact - Website- string- (optional) 

  • Contact - PhoneNumbers- Array - (optional) 

  • Contacts Array - (optional)
