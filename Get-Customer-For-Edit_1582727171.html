VDC : Get Customer For Edit

Created by Kavitha Thangappan, last modified on 11 nov., 2021

Customer who wants to edit existing customer details sends request to this endpoint.

System fetches the existing customer details from database using customer Id provided and map the details with the customer edit object.

For using this api, you need to be authorized.

Authentication

Please refer to this page. Authentication APIs - VDC - Confluence (atlassian.net)

Authorization

This requires api permission Pages_Vehicles

Value

Endpoint

​/api​/services​/app​/Customer/GetCustomerForEdit
