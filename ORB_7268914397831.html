1.1.1. Response format explanation

 

The response object returns the following object:

 

Property

In XML

Description

Voertuig

<voertuig>

Sent vehicle information

Transactie

<transactie>

Transaction data from the ORB request

 

The Response Data is an array of responses since there can be multiple actions done on a single endpoint.


 

Transactie property

The “transactie” property has the following properties

 

Property

In XML

Description

statuscode

<statuscode>

Response status code from RDW

 

kortemelding

<kortemelding>

I = information
W = warning

F = Failed

S = Application failure

extramelding

<extramelding>

Extra information from <kortemelding>

referentieidhen

<referentieidhen>

Reference of the request

Datumtijdregistratierdw

<datumtijdregistratierdw>

Date and tme registration RDW

transactiecoderdw

<transactiecoderdw>

TransactionCode from RDW

rdwbedrijfsnummer

<rdwbedrijfsnummer>

Company RDW number

bedrijfsnaam

<bedrijfsnaam>

Company name

vestigingsplaatsbedrijf

<vestigingsplaatsbedrijf>

City of company
