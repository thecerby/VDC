VDC : Get Documents

Created by Kavitha Thangappan, last modified on 11 nov., 2021

Customer who wants to fetch document details sends request to this endpoint.

System fetches all documents from document repository order by Id and each item with display order .

For using this api, you need to be authorized.

Authentication

Please refer to this page. Authentication APIs - VDC - Confluence (atlassian.net)

Authorization

 

Property

Value

Endpoint

​/api​/services​/app​/Document/GetDocuments

Method

GET

Authorization

Token {token from Authorized User}

Headers

Accept: application/json 

Content-Type: application/json 

Field Parameters

 

Input



Output
if request is successful , api returns below response. Response status 200
