VDC : Vehicle AFC daily export

Created by Stephan Oomen on 20 jan., 2022

On working days an export will be sent to the configured email address in the host settings that will contain the list of vehicles mentioning it will be sent to AFC. This is based on the value StorageLocation, based on the StorageLocationChanged the vehicle is added to the list.

For changing the e-mail address you can go to the Host account and go to Administration => Settings and then the tab “Schedules Tasks”
