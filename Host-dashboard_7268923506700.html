VDC : Host dashboard

Created by Stephan Oomen on 04 feb., 2022

The host dashboard is available when you you enable the permission Dashboard.

Within the dashboard we have a list of widgets that can be added, by enabling edit mode, you can add widgets, pages and fallback to default.

The default dashboard contains the following widgets:

  • KFZ- Zulassungsstelle statistics

KFZ- Zulassungsstelle statistics

Requires vehicles permission

This shows the KFZ- Zulassungsstelle statistics based on the number of vehicles placed in period:

  • Today

  • Yesterday

  • This week
