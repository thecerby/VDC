VDC : Kentekencardreader

Created by Stephan Oomen, last modified on 28 jul., 2023

This document is outdated, please check RegistratieXpert Kentekencardreader

KentekencardReader is een ‘deploy once’ win32 applicatie, die middels een USB aansluiting kentekencard chips kan uitlezen.

De libraries zijn ergens van een github betrokken, en worden niet door VDC/MPlus onderhouden. (Niets aan doen dus).

.NET versie is 4.5.2 omdat (op moment van schrijven) deze applicatie dus bij eindklanten terecht komen, het verzoek om 4.6.1 of hoger te installeren zal extra supportverzoeken genereren. 4.5.2 komt standaard mee met Windows 10.
