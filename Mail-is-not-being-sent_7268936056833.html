VDC : Mail is not being sent

Created by Stephan Oomen on 23 mrt., 2022

It is possible that a customer of VDC will complain about emails not being sent. Follow the below procedure to identify the issue.

 Instructions

To identify if this is an issue on server side, try the following

  • Go to the SQL Server and run the following query, this will return the records pending to be sent.

    SELECT TOP 100 * FROM VDC_RE.dbo.TblMailRequests 
WHERE PROCESS_ID IS NULL AND CREATED > dateadd(MM, -1, getdate())

If the above does return records, follow the below procedure:

  1. Go to the portal server and check for the SMTP handler

  2. Turn it off and start it with the shortcut found in the dashboard named SMTP Handler (Mail portal)

It will ask you for some parameters, just hit OK with the default values.

3. Re-run the query to see if the number of records will drop, this could take a couple of minutes.

If all is looking OK, check with the customer on which address the email is sent to and check in Sendgrid if the email is sent.
