VDC : Overzicht VDC situatie 2015

Created by Stephan Oomen on 01 dec., 2021

Servers Productie:

 

Productie:           VDC-PROD-PORTLS.cloudapp.net:3389  

WEB:                     VDC-PROD-WEB.cloudapp.net:60912

SQL:                       VDC-PROD-SQL.cloudapp.net:81

 

Server Testomgeving:

 

Portalen:             VDC-TEST-PORTLS.cloudapp.net:55300

WEB:                     VDC-TEST-WEB.cloudapp.net

SQL:                       VDC-TEST-SQL.cloudapp.net

De servers zijn opgezet in een Microsoft Azure omgeving.

 

Het landschap bestaat uit de volgende onderdelen:

 

  • Webserver (Serveert Silverlight app, Diverse webservices, Klant documenten)

  • SQL server (MSSQL, Reporting server)

  • Portal server (RDW Klant portalen, Email afhandeling, Batch verwerking)

  • Backoffice applicatie welke door VDC backoffice wordt gebruikt

  • Diverse externe (third party) webservices als RDW OVI, VIES (BTW Check), en A2SP

 In hoofdlijnen bestaat de applicatie uit een Silverlight applicatie die draait in de browser van de eindgebruiker. Deze applicatie maakt verbinding met een webserver die op zijn beurt gegevens ophaalt en wegschrijft in een database. Tenslotte draaien er diverse ‘portaal’ applicaties op een derde server die namens de eindgebruiker acties uitvoert op het web portaal van de RDW.

 

Per server uitgelicht wat erop draait:

 

  • Productie server: VDC-PROD-PORTLS.cloudapp.net:3389

  • Klant Portalen

     Te weten, ORB, OREH, Kcb, Machting, Tdn , Tv, Ored, Mail

  • Klant RDW certificaten

  • Batch routines tool

  • SMTP Handler

  • RegistratieXpert BackOffice. Dit zal binnenkort een clickonce installatie worden via https://vdc.registratiexpert.nl/backoffice/

 

  • Web server: VDC-PROD-WEB.cloudapp.net:60912

  • IIS

  • registratieXpert.nl  https://vdc.registratiexpert.nl

  • Cobian Backup (maakt een backup van alle documenten en applicatiefiles)

  • Service voor Resetten wachtwoord van RDWovi (Windows Service: VDC RDW Password Reset Service)

  • BCA Service. Dit is feitelijk de eerste Soap server versie van de gehele applicatie, alleen Autoveiling wordt hiermee afgehandeld.

  • External Services. Deze worden aangeroepen door RegistratieXpert BackOffice en in enkele gevallen door RegistratieXpert Services (pass through)

  • Mail Services: Deze service gebruikt ‘sendgrid’ (cloud based) om SMTP mail te versturen

  • RegistratieXpert Services.  Voornamelijk aangeroepen door de Silverlight applet en door de Portals die per seconde pollen of er een nieuwe opdracht moet worden verwerkt.

 

  • SQL server: VDC-PROD-SQL.cloudapp.net:81

  • Wachtwoord reset tool

  • Reporting (onder andere BCA mail met werkzaamheden van de dag ervoor)

  • Database (SQL Server 2014 SP1)

  • Management Studio SQL 2014 

Binnen Azure is er een extra VHD (Schijf) aangemaakt en is gekoppeld aan de Productie server (drive letter E) voor backups.

 

Client installatie (Optioneel voor wie deze module wenst)

  1. Kentekencard reader. Dit is een .NET 4.5.1 clickonce applicatie.

Installatie geschiedt eenmalig via https://vdc.registratiexpert.nl/kentekencardreader/ . Updates worden automatisch verspreid/
